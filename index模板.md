# Index 生成指南

## 用途

本模板用于为按 `[¶XXXX]` 段落号标注的 Markdown 教材生成结构化索引文件。索引文件为课程讲解、练习生成和掌握追踪提供段落级定位基础。

## 生成步骤

1. **扫描章节文件**：遍历 `chapters/` 目录下所有 `.md` 文件
2. **统计段落数**：计算每个文件中 `[¶XXXX]` 标记的总数（即最大 XXXX 值）
3. **统计图片数**：计算每个文件中 `![](img/...)` 标记的总数
4. **填充文件导航表**：按上述统计结果填写 `## 文件导航` 表格
5. **生成路由表**：为每个章节生成 `## 第X章路由` 表格，将段落号范围映射到教学单元

## 路由表生成规则

路由表将每个章节的段落号划分为连续区间，每个区间对应一个教学单元：

- **单元标识**（Unit）：格式为 `{chapter}-{section}[-{subsection}]`，如 `ch01-03-02`
- **段落范围**（Range）：格式为 `¶XXXX-¶YYYY`，表示该单元覆盖的正文段落区间
- **注释**（Note）：标注对应的小节编号或内容摘要
- **特殊规则**：
  - 每章的 `¶0001`~`¶00XX` 通常为引言/导读，标识为 `{chapter}-00`
  - 每章末尾的参考文献段落不进入路由表，仅在注释中标注 `references start at ¶XXXX`
  - 标题行不算段落，路由表引用的是正文段落号

## 格式约束

- 标题行（`##`、`###` 等）不占用段落号，不纳入索引统计
- 只有正文段落、表格、图片块和公式块使用 `[¶XXXX]` 编号
- 段落号全局连续，从 `0001` 开始
- 图片链接统一使用 `img/...` 相对路径

## 使用规则

1. 课程讲解、练习和掌握追踪必须回指本索引中的正文段落号。
2. 标题行不是可引用段落；需要引用小节时使用该小节的正文范围。
3. 图片链接统一使用 `img/...`，Markdown 可直接渲染。
