## 本节教学目标

学完本节后，学习者应该能够：

- 写出 SVM 的对偶优化问题形式。
- 理解为什么 SVM 的决策函数只依赖于支持向量。
- 解释"强对偶性"和"互补松弛条件"在 SVM 中的作用。

## 知识点分层

### 核心知识点

| 知识点 | 原文依据 | 讲解要求 | 出题权限 |
|---|---|---|---|
| SVM 主问题（Primal）的凸优化形式：min (1/2)||w||^2, s.t. 1 - y^(n)(w^T x^(n) + b) ≤ 0, ∀n | `¶0272-¶0273` | 必须讲清：这是从最大化间隔自然得到的凸二次规划问题。 | 正式题 |
| 拉格朗日函数：Λ(w, b, λ) = (1/2)||w||^2 + Σ λ_n(1 - y^(n)(w^T x^(n) + b))，其中 λ_n ≥ 0 | `¶0274-¶0276` | 讲清：引入拉格朗日乘数 λ_n 将约束纳入目标函数。 | 正式题 |
| 对偶问题的意义：(1) 可用核函数（下单元）；(2) 解只依赖于支持向量 | `¶0282` | 讲清"为什么要求对偶"的动机——引入核函数的入口。不要求完整推导对偶函数。 | 正式题 |
| 支持向量与 λ_n 的关系：根据 KKT 互补松弛条件，λ_n* > 0 当且仅当 x^(n) 是支持向量（在间隔边界上）；非支持向量的 λ_n* = 0 | `¶0283` | 必须讲清："只有支持向量对应的 λ_n* > 0"，因此 SVM 的解是稀疏的。 | 正式题 |
| SVM 决策函数：f(x) = sgn(Σ λ_n* y^(n) (x^(n))^T x + b*)，只依赖于支持向量（λ_n* > 0 的样本） | `¶0285-¶0290` | 讲清：决策函数是对偶解的直接应用，且支持向量通常远少于总样本数。 | 正式题 |
