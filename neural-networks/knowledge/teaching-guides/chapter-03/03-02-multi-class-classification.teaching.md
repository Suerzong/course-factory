## 本节教学目标

学完本节后，学习者应该能够：

- 说出多分类的三种处理方式及其名称。
- 区分"一对其余""一对一""argmax"三种方式的判别函数数量和决策方式。
- 解释为什么"argmax"方式解决了前两种方式的"难确定区域"缺陷。
- 复述定义3.2（多类线性可分）的完整表述。

| 知识点 | 原文依据 | 讲解要求 | 出题权限 |

| "一对其余"方式：C 个二分类问题，每个判别函数将类别 c 与其余类分开 | `¶0035` | 讲清判别函数数量和每个函数的作用范围。 | 正式题 |

| "argmax"方式：C 个判别函数，取最大判别的类别（公式 (3.10)-(3.11)） | `¶0038-¶0041` | 必须讲清 f_c = w_c^T x + b_c，然后 y = argmax_c f_c。 | 正式题 |

| 定义3.2 — 多类线性可分：存在 C 个 w_c*，使第 c 类所有样本的 f_c 超过其他所有类的 f_~c | `¶0047` | 与定义3.1对比：两类是单一 w*，多类是 C 个 w_c* 两两比较。 | 正式题 |
