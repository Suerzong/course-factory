## 诊断目标

完成本模块补救后，学习者应该能够：

- 写出 Logistic 函数（Sigmoid）的导数公式：σ'(x) = σ(x)(1−σ(x))。
- 写出 Softmax 函数的定义：p_i = exp(s_i)/∑_j exp(s_j)。
- 写出 Softmax 的雅可比矩阵元素：∂p_i/∂s_j = p_i(δ_{ij} − p_j)。
- 推导并记住交叉熵 + Softmax 组合的简化梯度：∂L/∂s_i = p_i − y_i。

## 知识点分层

### 核心知识点

| 知识点 | 原文依据 | 讲解要求 | 出题权限 |
|---|---|---|---|
| Logistic 函数（Sigmoid）：σ(x) = 1/(1+e^{-x})，导数 σ'(x) = σ(x)(1−σ(x)) | `¶0237-¶0258` | 展示 S 型曲线形状和值域 (0,1)。推导导数公式：从 σ' = σ(1−σ) 一步到位，强调"用函数值表达导数"的简洁性。这是 NNDL 中 Sigmoid 激活函数反向传播的基础。 | 课堂识别不计分 |
| Softmax 函数定义：p_i = exp(s_i)/∑_j exp(s_j)，将任意实数向量映射为概率分布 | `¶0259-¶0269` | 说明 Softmax 的作用：把"分数"（logits）变成概率（非负、和为 1）。用三分类的具体数字例子计算一遍。 | 课堂识别不计分 |
| Softmax 的雅可比矩阵：∂p_i/∂s_j = p_i(δ_{ij} − p_j)，其中 δ_{ij} 是 Kronecker delta | `¶0261-¶0269` | 分两种情况讲：i=j 时 ∂p_i/∂s_i = p_i(1−p_i)；i≠j 时 ∂p_i/∂s_j = −p_i p_j。强调雅可比矩阵不是对角的（各分量之间有竞争关系）。 | 课堂识别不计分 |
| 交叉熵 + Softmax 的简化梯度：∂L/∂s_i = p_i − y_i（当损失函数为交叉熵 L = −∑ y_i log p_i 时） | —（推导在 B.4.2.2 和 E.3 中） | 这是 NNDL 中最常用的输出层梯度公式。说明这个极简形式是"交叉熵和 Softmax 的指数求导恰好抵消"的结果。学习者必须记住这个结果。 | 课堂识别不计分 |
