## 诊断目标

完成本模块补救后，学习者应该能够：

- 用矩阵乘法表示两个线性映射的复合，理解维度匹配规则。
- 写出转置的性质：(AB)ᵀ = BᵀAᵀ。
- 解释逆矩阵的定义和存在条件（方阵且行列式非零）。
- 识别正交矩阵的定义性性质：QᵀQ = I，列向量为单位正交基。
- 解释对角矩阵如何实现各维度的独立缩放。
- 用矩阵形式写出仿射变换 y = Wx + b。

## 知识点分层

### 核心知识点

| 知识点 | 原文依据 | 讲解要求 | 出题权限 |
|---|---|---|---|
| 矩阵乘法的定义与维度匹配：A_{m×n} B_{n×p} = C_{m×p}，C 的每个元素是 A 的行与 B 的列的内积 | `¶0066-¶0070` | 通过具体数字矩阵手算一次乘法，强调"内积"视角。讲清维度不匹配时乘法无定义。 | 课堂识别不计分 |
| 转置的性质：(AB)ᵀ = BᵀAᵀ（注意顺序反转） | `¶0074-¶0075` | 用 2×2 矩阵验证该性质，强调反转顺序是常见错误点。 | 课堂识别不计分 |
| 逆矩阵的定义：A A⁻¹ = I，仅方阵且行列式非零时可逆 | `¶0112-¶0115` | 用 2×2 矩阵公式 A⁻¹ = (1/det(A)) [[d, -b], [-c, a]] 直观展示。说明不可逆意味着存在"信息丢失"（降维）。 | 课堂识别不计分 |
| 正交矩阵：QᵀQ = I，即 Q⁻¹ = Qᵀ，列（行）向量为单位正交基 | `¶0119-¶0122` | 解释正交矩阵代表"旋转/反射"变换（保长度、保角度）。连接到 NNDL 中正交初始化的动机。 | 课堂识别不计分 |
| 对角矩阵：只有主对角线非零，diag(a) x = a ⊙ x（逐元素缩放） | `¶0102-¶0108` | 直观：对角矩阵对向量每一维独立缩放，不同维度之间无交互。连接到 NNDL 中学习率逐参数调整。 | 课堂识别不计分 |
| 仿射变换的矩阵表示：y = Wx + b（线性变换 + 平移） | `¶0059-¶0062` | 这是 NNDL 全连接层的标准形式，必须讲清 W 和 b 各自的角色。可以用二维图形展示：W 负责旋转/缩放，b 负责整体平移。 | 课堂识别不计分 |

| 内容 | 原文依据 | 处理方式 |

- [ ] 学习者能用矩阵形式写出 y = Wx + b，并说明 W 和 b 各自的作用。
