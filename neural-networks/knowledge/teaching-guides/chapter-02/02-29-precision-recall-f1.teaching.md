## 本节教学目标

学完本节后，学习者应该能够：

- 写出精确率和召回率的公式。
- 解释精确率和召回率的权衡关系。
- 写出 F 值的公式并解释 β 参数的作用。

## 知识点分层

### 核心知识点

| 知识点 | 原文依据 | 讲解要求 | 出题权限 |
|---|---|---|---|
| 精确率（查准率）𝒫_c = TP_c / (TP_c + FP_c)：预测为 c 的样本中实际是 c 的比例 | `¶0343-¶0344` | 用"抓坏人"类比：抓的人中有多少是真正的坏人。 | 正式题 |
| 召回率（查全率）ℛ_c = TP_c / (TP_c + FN_c)：实际为 c 的样本中被预测为 c 的比例 | `¶0346-¶0347` | 用"抓坏人"类比：真正的坏人中有多少被抓到了。 | 正式题 |
| F值：精确率和召回率的调和平均，ℱ_c = (1+β²)·𝒫_c·ℛ_c / (β²·𝒫_c + ℛ_c) | `¶0348-¶0349` | 讲清 β=1 时就是 F1（精确率和召回率同等重要），β>1 时召回率更重要。 | 正式题 |

### 重要背景

| 内容 | 原文依据 | 处理方式 |

| β 用于平衡精确率和召回率的重要性 | `¶0350` | 通常 β=1。 |
