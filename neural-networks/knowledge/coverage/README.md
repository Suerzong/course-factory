# Coverage

本目录保存知识点覆盖审计记录，用来确认学习路线、教学指引、教材段落、练习边界和掌握追踪之间没有孤岛。

## 当前文件

- `chapter-01-coverage.md`：第1章覆盖审计表，按单元串联 `单元ID -> 讲义 -> 教材段落 -> 核心知识点 -> 可出题范围 -> 掌握追踪行 -> 下一单元`。

## 使用规则

- 新增或调整学习路线时，必须同步更新覆盖审计表。
- 每个可教学单元必须有对应教学指引、教材段落和掌握追踪行。
- 只有 `出题权限=正式题` 的核心知识点可以进入正式练习、章测和推进依据。
- `课堂识别不计分` 只能用于轻量确认，`了解即可` 不进入正式考核。
