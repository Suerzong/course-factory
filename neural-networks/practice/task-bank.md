# NNDL 训练任务库

本任务库只收录当前 NNDL 主课任务。每个任务应在 10-20 分钟内完成，目标是读懂教材、解释概念、理解公式和完成课本题。

## 任务模板

```md
## 任务名称

- NNDL 位置：
- 对应单元：
- 考察来源：核心知识点 / 重要背景 / 已学前置 / 课本习题
- 教材段落：
- 知识点层级：核心知识点 / 重要背景 / 了解即可
- 出题权限：正式题 / 课堂识别不计分 / 否
- 未学内容检查：
- 目标：
- 课本依据：
- 任务要求：
- 成功标准：
- 常见误解：
- 反馈重点：
```

## 课本习题处理规则

课本习题不直接整题丢给学习者。应按以下步骤拆解：

1. 标明习题编号和对应教材小节。
2. 判断习题依赖的概念是否已讲。
3. 如果依赖后文或外部文献，先降级为“课堂识别不计分”或推迟。
4. 对核心习题拆成“题意理解 -> 所需概念 -> 关键一步 -> 完整作答”。
5. 只把已学范围内的部分作为正式考核。
