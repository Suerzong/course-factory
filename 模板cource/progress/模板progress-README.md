# 学习进度说明

`progress/` 保存当前课程推进状态。它只记录"讲到哪里、掌握到什么程度、是否允许推进"，不保存教材原文，也不替代知识点指引。

## 文件职责

- `current-position.md`：当前学习入口，AI 每次开始教学前先读这个文件。
- `mastery-tracker.md`：掌握度追踪表。

## 掌握度追踪表格式

`mastery-tracker.md` 的章节表统一使用以下表头：

| 单元ID | 技能 | {{COURSE_SHORT_NAME}} 章节 | 等级 | 正确率 | 最近测试 | 最近证据 | 主要弱点 | 推进 |
|---|---|---|:---:|:---:|---|---|---|:---:|

说明：

- `单元ID` 必须和 `learning-path/chapter-XX.md` 中的单元 ID 完全一致。
- `技能` 写学习者需要掌握的能力或主题。
- `{{COURSE_SHORT_NAME}} 章节` 写教材章节、小节或章测范围。
- `等级` 使用 0-5 掌握度等级。
- `正确率` 记录最近一次诊断、练习或章测结果。
- `最近测试` 记录日期、测试类型或 `-`。
- `最近证据` 记录最近一次学习日志、题目编号、练习来源或 `-`。
- `主要弱点` 只记录最影响推进的薄弱点。
- `推进` 只写 `是` 或 `否`。
- 掌握度表中的 `单元ID`、`progress/current-position.md` 和教学指引必须互相一致。

字段硬约束：

- `推进` 字段只能写 `是` 或 `否`，不得写 `是 -> chXX`、说明文字或下一单元。
- `正确率` 不得把初测、练习和再测折算成一个新百分比；若经历补救，应写成 `67% -> 再测100%` 这类证据链。
- `最近证据` 必须指向学习日志文件和关键结果，例如 `logs/learning-sessions/2026-05-23-ch01-00.md；练习67% -> 再测100%`。

## 状态含义

| 状态 | 含义 |
|---|---|
| 待重校准 | 旧记录存在，但需要按新知识点指引重新确认 |
| 未开始 | 尚未按新路线学习 |
| 诊断中 | 正在判断当前水平和薄弱点 |
| 学习中 | 正在讲解或练习该单元 |
| 待检测 | 已讲完，等待诊断或练习确认 |
| 补救中 | 未达标，正在针对弱点补讲或补练 |
| 再测中 | 补救后等待再次检测 |
| 已通过 | 满足推进条件 |
| 暂缓 | 暂不推进，需补讲或补练 |

`当前状态` 只能使用上表中的状态值。不得写 `待诊断`、`通过中`、`已完成` 等未列出的自由状态。推进到下一单元后，下一单元默认状态为 `未开始`；真正开始诊断时才改为 `诊断中`。

## 状态流转

| 当前状态 | 触发条件 | 目标状态 |
|---|---|---|
| 待重校准 | 重新诊断完成 | 诊断中 |
| 未开始 | AI 开始诊断 | 诊断中 |
| 诊断中 | 诊断完成，进入讲解 | 学习中 |
| 学习中 | 讲解完成，等待练习确认 | 待检测 |
| 待检测 | 练习正确率 ≥90% 且核心题通过 | 已通过 |
| 待检测 | 练习正确率 80%-89% | 补救中 |
| 待检测 | 练习正确率 <80% | 补救中 |
| 补救中 | 补救讲解完成，等待再测 | 再测中 |
| 再测中 | 再测正确率 ≥90% 且核心题通过 | 已通过 |
| 再测中 | 再测仍未达标 | 暂缓 |
| 暂缓 | 重新补救 | 补救中 |
| 已通过 | 进入下一单元 | 未开始（下一单元） |

不允许的流转：

- `未开始` → `已通过`：不得跳过诊断和检测。
- `学习中` → `已通过`：不得跳过检测。
- `待检测` → `未开始`：不得直接重置，必须走补救或通过。
- `再测中` → `学习中`：不得回退到讲解，只能走补救或暂缓。
- `暂缓` → `已通过`：不得跳过补救和再测。

## 推进规则

- 同单元核心知识点正确率低于 80%：不推进。
- 80%-89%：可继续同单元补弱点，不解锁下一单元。
- 90% 以上且核心题通过：可推进到下一单元。
- 章测通过后才解锁下一章。
- 工具概览、总结和深入阅读可以轻量检测，但不能替代核心概念单元。
- 未达标时，状态应进入 `补救中` 或 `再测中`，不能直接推进。

## 使用规则

- AI 开始教学前先读 `current-position.md`。
- AI 应按 `mastery-loop.md` 执行诊断、讲解、练习、反馈、补救和再测。
- AI 讲解单元前必须读取对应 `learning-path/chapter-XX.md`。
- AI 更新掌握度时必须保留教材段落依据、最近测试记录和最近证据。
- AI 更新 `current-position.md` 时必须同步更新顶部字段、教学入口和当前推进要求；不得留下上一单元的教学指引、教材段落或下一单元。
