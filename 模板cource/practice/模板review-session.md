# {{COURSE_SHORT_NAME}} 间隔复习规则

## 触发条件

每次新会话开始时，AI 检查 `review/concept-cards.md`：

- 若存在 `下次复习 ≤ 今天` 的卡片 → 先复习，再学新内容。
- 若无到期卡片 → 直接进入新内容。

## 复习出题规则

- 每张到期卡片出 1 题回忆题。
- 按卡片类型选择对应题型：
  - **公式**：AI 给出公式，学生解释每个符号的含义和公式的作用。
  - **概念**：用自己的话复述定义，不背原文。
  - **关系**：说明 A 和 B 的区别或联系。
  - **流程**：按顺序回忆关键步骤。
  - **结构**：描述组成部分及各部分作用。
- 所有题型均为口头回答，不要求学生输入公式或代码。
- 不需要五行标注（复习题不是正式练习题）。
- 每次复习最多 5 张卡片，避免疲劳。

## 复习判定

- 回忆正确（能说出定义/公式/区别）：连续正确 +1，按间隔表推进。
- 回忆错误（说不清或说错）：连续正确归零，回到 +1 天间隔。
- 部分正确：标记为"需巩固"，间隔不推进，下次 +1 天再复习。

## 间隔表

| 连续正确次数 | 下次复习间隔 |
|---:|---|
| 0 | +1 天 |
| 1 | +2 天 |
| 2 | +4 天 |
| 3 | +7 天 |
| 4 | +15 天 |
| 5+ | +30 天 |
